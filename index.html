Notes

Linux — CPU & process monitoring

# Real-time process overview
top -d 1

# Per-CPU usage breakdown
mpstat -P ALL 1 5

# Top CPU consumers
ps aux --sort=-%cpu | head -15

# Kernel-level CPU profile
perf top -a -g

# Scheduler statistics
cat /proc/schedstat

# I/O wait per process
iotop -o

Networking — capture & diagnose

# Capture TCP SYN packets on eth0
tcpdump -i eth0 'tcp[tcpflags] & tcp-syn != 0'

# Capture HTTP traffic (plain text)
tcpdump -i any -A 'tcp port 80'

# Active connections + PIDs
ss -tulnp

# Legacy alternative
netstat -tulnp

# Routing table
ip route show

# Trace path (no DNS resolution)
traceroute -n 8.8.8.8

# Test TCP port without Telnet
nc -zv 192.168.1.1 443

TCP three-way handshake & teardown (ASCII)

  CLIENT                              SERVER
    |                                   |
    |  ------  SYN  (seq=x)  ------->  |   [SYN_SENT]
    |                                   |   [SYN_RCVD]
    |  <-- SYN-ACK (seq=y, ack=x+1) -- |
    |                                   |
    |  ------  ACK  (ack=y+1)  ----->  |   [ESTABLISHED]
    |                                   |   [ESTABLISHED]
    |  ========= DATA FLOW ==========> |
    |  <========= DATA FLOW ==========  |
    |                                   |
    |  ------  FIN  ----------------->  |   [FIN_WAIT_1]
    |  <-----  ACK  ------------------  |   [CLOSE_WAIT]
    |  <-----  FIN  ------------------  |   [LAST_ACK]
    |  ------  ACK  ----------------->  |   [TIME_WAIT]
    |                                   |   [CLOSED]
    | (waits 2×MSL, then CLOSED)        |

Security — quick reference

# Open ports and owning process
ss -tulnp | grep LISTEN

# Firewall rules (iptables)
iptables -L -n -v --line-numbers

# Find SUID binaries (priv-esc surface)
find / -perm -4000 -type f 2>/dev/null

# Watch failed SSH logins live
tail -f /var/log/auth.log | grep 'Failed password'

# Integrity check
sha256sum /path/to/binary

# Network connections by process
lsof -i -n -P

Applied math — graph Laplacian (Python / NumPy)

import numpy as np

# Adjacency matrix: graph  0 -- 1 -- 2
A = np.array([[0,1,0],
              [1,0,1],
              [0,1,0]])

D = np.diag(A.sum(axis=1))   # degree matrix
L = D - A                    # graph Laplacian

vals = np.linalg.eigvalsh(L)
print(vals)
# → [~0.   1.586   2.414]
# Fiedler value (2nd) ≈ 1.586 → well-connected graph

# Eigenvector centrality via dominant eigenvector
evals, evecs = np.linalg.eig(A.T)
pr = np.abs(evecs[:, np.argmax(evals)])
print(pr / pr.sum())   # normalized scores per node
